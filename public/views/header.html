rummy.score

rummy.score is an app for keeping track of your rummy 500 games.

Click 'begin' and then select the number of players (2-4) in your game from the dropdown.

A list of all available players will appear; select them using shift-click or control-click to make multiple selections. A new player can be created using the appropriate form.

When you're satisfied with the player selection, click the button and then, after the first turn in your game is complete, click the 'let's score' button to start entering scores. The game will automatically track and total the scores for you, and indicate when someone has won.

That's it! It's pretty simple. rummy.score is a project of Ethan Friedman.

rummy.score was developed as a project by Ethan Friedman as an experiment in building a MEAN stack app

What's a MEAN stack you ask?

MongoDB: MongoDB is a noSQL database used to store the player and game data.

Express.js: Express.js is a web server application framework for Node.js (see below).

AngularJS: AngularJS is a front-end framework used for building single-page applications such as this one. It provides two-way data binding between the view on the page and the model behind the scenes.

Node.js: Node.js is a server-side runtime environment for building web applications. It comes with a huge library of packages to make building web apps fast and seamless.
